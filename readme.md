# CertiChain – Certificate Verification using Algorand

## 📌 Project Description
CertiChain is a blockchain-based certificate verification system built using Algorand. 
It helps institutions issue tamper-proof digital certificates and allows anyone to verify them securely.

## 🚀 Features
- Upload and store certificate data on Algorand blockchain
- Generate unique certificate ID
- Verify certificates easily
- Prevent fake certificates

## 🛠️ Technologies Used
- Algorand Blockchain
- Python (PyTeal / SDK)
- GitHub for version control

## 📂 Project Status
This project is in initial development stage.
